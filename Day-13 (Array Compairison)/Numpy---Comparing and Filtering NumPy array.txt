			========================================================
						Numpy---Comparing and Filtering NumPy array
			========================================================
=>For Comparing NumPy Array we use all Relational Operator.
=>Along with Relational Operators, we can also Use Realtional Functions Present in numpy module.

		Greater than (>) OR    numpy.greater().
		Less Than (<)	    OR    numpy.less().
		Equal(==)	    OR    numpy.equal()
		Not Equal(!=)      OR    numpy.not_equal().
		Greater than and equal to(>=).
		Less than Equal to(<=).
---------------------------------------------------------------------------------------------------------------------------------------------
Example 1:
---------------------------------------------------------------------------------------------------------------------------------------------
import numpy as np 
# Creating Array
a = np.array([1,2,3,4]) 
b = np.array([3,8,5,6])
# Comparing two arrays
np.greater(a, b)
---------------------------------------------------------------------------------------------------------------------------------------------
Output:  array([False, False, False, False])
---------------------------------------------------------------------------------------------------------------------------------------------
Example 2:
---------------------------------------------------------------------------------------------------------------------------------------------
import numpy as np
# Creating Array using NumPy
a = np.array([1, 2, 3, 4])
b = np.array([3, 8, 5, 6])
np.less(a, b)
---------------------------------------------------------------------------------------------------------------------------------------------
Output:  array([ True,  True,  True,  True])
---------------------------------------------------------------------------------------------------------------------------------------------
Example 3:
import numpy as np
a = np.array([1, 2, 3, 4])
b = np.array([3, 8, 5, 6])
np.equal(a, b)
---------------------------------------------------------------------------------------------------------------------------------------------
Output: array([ False,  False,  False, False])
---------------------------------------------------------------------------------------------------------------------------------------------
Example 4:
---------------------------------------------------------------------------------------------------------------------------------------------
import numpy as np
# Create Arrays using np.array() Function.
a = np.array([1, 2, 3, 4])
b = np.array([3, 8, 5, 6])
np.not_equal(a, b)
---------------------------------------------------------------------------------------------------------------------------------------------
Output: array([ True,  True,  True,  True])
---------------------------------------------------------------------------------------------------------------------------------------------
